公共財ゲームの説明

このゲームは、{{ C.PLAYERS_PER_GROUP }}人の参加者で行います。各参加者は、初期に{{ C.ENDOWMENT }}ポイントを持っています。

ゲームの流れは以下の通りです：

  1. 各参加者は、自分のポイントの一部または全部を「公共財」に貢献します。貢献しないポイントは自分のものとして残ります。
  2. 全員の貢献額の合計を{{ C.MULTIPLIER }}倍したポイントが、公共財として作成されます。
  3. この公共財は、{{ C.PLAYERS_PER_GROUP }}人の参加者で等しく分けられます。つまり、各参加者は公共財から等しいシェアを受け取ります。

報酬の計算例：
仮に{{ C.PLAYERS_PER_GROUP }}人が全員{{ C.ENDOWMENT }}ポイント貢献した場合：
貢献額合計 = {{ C.PLAYERS_PER_GROUP }} × {{ C.ENDOWMENT }} = 60
公共財 = 60 × {{ C.MULTIPLIER }} = 96
各参加者のシェア = 96 ÷ {{ C.PLAYERS_PER_GROUP }} = 32
各参加者の総報酬 = 残ったポイント + シェア = 0 + 32 = 32

貢献額を減らすと、自分の残りポイントは増えますが、公共財が減り、全員のシェアが減る可能性があります。貢献額を増やすと、自分の残りポイントは減りますが、公共財が増え、全員のシェアが増える可能性があります。

あなたの目標は、自分の総報酬を最大化することです。他の参加者の貢献額はわかりません。
